# graph-olap-schemas

Shared Pydantic schemas for the Graph OLAP Platform.

## Installation

```bash
pip install graph-olap-schemas
```

## Usage

```python
from graph_olap_schemas import InstanceSchema, MappingSchema, SnapshotSchema
```

## Development

```bash
make install    # Install in editable mode with dev dependencies
make test       # Run tests
make lint       # Run linter
make build      # Build wheel
```
