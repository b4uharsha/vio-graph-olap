# Tutorials

Step-by-step learning tracks for Graph OLAP.

## Tracks

| Track | Description | Notebooks | Time |
|-------|-------------|-----------|------|
| **00-start-here** | Welcome and quick start | 2 | 10 min |
| **01-fundamentals** | SDK basics: clients, resources, queries | 8 | 2 hours |
| **02-cypher** | Cypher query language | 8 | 2 hours |
| **03-algorithms** | Graph algorithms: centrality, community, paths | 23 | 3 hours |
| **04-admin-ops** | Administration and operations | 4 | 45 min |
| **05-advanced** | Instance lifecycle, exports, engines | 5 | 1.5 hours |

## Recommended Order

1. Start with **00-start-here** for orientation
2. Complete **01-fundamentals** for SDK proficiency
3. Learn **02-cypher** for query capabilities
4. Explore **03-algorithms** for analytics
5. Finish with **04-admin-ops** and **05-advanced**

## Prerequisites

- Access to Graph OLAP JupyterHub
- Basic Python knowledge
- SQL familiarity helpful
