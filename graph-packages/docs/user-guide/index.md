## What You Can Do

- **Create mappings** from Starburst data sources
- **Provision graph instances** for analysis
- **Run Cypher queries** to explore relationships
- **Execute graph algorithms** (PageRank, community detection, pathfinding)
- **Export results** to various formats (DataFrame, NetworkX, dict)
- **Visualize graphs** directly in Jupyter notebooks

## Prerequisites

- Python 3.10+ with Jupyter environment
- Access to the Graph OLAP platform (API key or network access)
- Basic familiarity with SQL and Python
