## Platform Capabilities

| Capability | Description |
|------------|-------------|
| **Graph Instance Management** | Create and manage graph database instances from Starburst queries |
| **Data Export Pipeline** | Automated export from Starburst to graph databases via Parquet |
| **Graph Algorithms** | PageRank, community detection, shortest path, and more |
| **Multi-Engine Support** | RyuGraph (KuzuDB) and FalkorDB backends |
| **Python SDK** | Full-featured SDK for Jupyter notebook integration |
